## Sample API Calls

### List Exchanges

Discover which exchanges your API key has access to:

<CodeGroup>

```bash cURL
curl -X GET https://api.rulebookcompany.com/api/v1/exchanges/ \
  -H "x-rulebook-api-access-key: YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.rulebookcompany.com/api/v1/exchanges/"
headers = {
    "x-rulebook-api-access-key": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript JavaScript
const url = "https://api.rulebookcompany.com/api/v1/exchanges/";
const headers = {
  "x-rulebook-api-access-key": "YOUR_API_KEY",
};

fetch(url, { headers })
  .then((response) => response.json())
  .then((data) => console.log(data));
```

</CodeGroup>

### List Fee Schedule Results

Query fee schedule data with filters, sorting, and pagination:

<CodeGroup>

```bash cURL
curl -X GET "https://api.rulebookcompany.com/api/v1/fee-schedule-results/?exchange_name=Nasdaq&fee_type=OPTION&fee_action=Make&latest_only=true&page_size=5" \
  -H "x-rulebook-api-access-key: YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.rulebookcompany.com/api/v1/fee-schedule-results/"
params = {
    "exchange_name": "Nasdaq",
    "fee_type": "OPTION",
    "fee_action": "Make",
    "latest_only": True,
    "page_size": 5
}
headers = {
    "x-rulebook-api-access-key": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const params = new URLSearchParams({
  exchange_name: "Nasdaq",
  fee_type: "OPTION",
  fee_action: "Make",
  latest_only: "true",
  page_size: "5",
});

const url = `https://api.rulebookcompany.com/api/v1/fee-schedule-results/?${params}`;
const headers = {
  "x-rulebook-api-access-key": "YOUR_API_KEY",
};

fetch(url, { headers })
  .then((response) => response.json())
  .then((data) => console.log(data));
```
