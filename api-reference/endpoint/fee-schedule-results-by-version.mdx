## Overview

Returns all fee schedule results belonging to a specific extraction version. Each version represents a single extraction run against a fee schedule document. Use this to retrieve the complete set of fees from that version.

Results are access-scoped — you can only retrieve versions for exchanges associated with your API key.

<Note>
  **Authorization Required** - Include your API key in the
  `x-rulebook-api-access-key` header. Contact
  [aaron@rulebookcompany.com](mailto:aaron@rulebookcompany.com) to obtain
  access.
</Note>

<Tip>
  You can find `version_id` values in the response of the [List Fee Schedule
  Results](/api-reference/endpoint/fee-schedule-results) or [Get Fee Schedule
  Result](/api-reference/endpoint/fee-schedule-result-detail) endpoints.
</Tip>

## Path Parameters

<ParamField path="version_id" type="string (uuid)" required>
  Fee extraction version identifier. Obtain this from the `version_id` field in
  fee schedule results.
</ParamField>

## Query Parameters

<ParamField query="page_size" type="integer" default="20">
  Records per page (1–100)
</ParamField>

<ParamField query="page_number" type="integer" default="0">
  Page number (0-indexed)
</ParamField>

<ParamField query="include_footnotes" type="boolean" default="false">
  When `true`, includes resolved footnotes from the source document in the `relationships` field of each result.
</ParamField>

<ParamField query="include_definitions" type="boolean" default="false">
  When `true`, includes resolved definitions from the source document in the `relationships` field of each result.
</ParamField>

<ParamField query="include_source" type="boolean" default="false">
  When `true`, includes `source_document`, `source_link`, and `page_number` fields in each result.
</ParamField>

## Request

```bash
curl -X GET "https://api.rulebookcompany.com/api/v1/fee-schedule-results/versions/f7e8d9c0-b1a2-3456-7890-abcdef123456" \
  -H "x-rulebook-api-access-key: YOUR_API_KEY"
```

### With Pagination and Source

```bash
curl -X GET "https://api.rulebookcompany.com/api/v1/fee-schedule-results/versions/f7e8d9c0-b1a2-3456-7890-abcdef123456?page_size=50&page_number=0&include_source=true" \
  -H "x-rulebook-api-access-key: YOUR_API_KEY"
```
